# How to run
## Alphaa (ERC1155)
```
node scripts/phoenix/alphaa.js ${token_id} ${your_account} ${amount}
```
example
```
node scripts/phoenix/alphaa.js 1 0x640a6D5A3f155A8F0636a0396B18Ba5eEdfab440 1
```

## AlphaaIO (ERC721)
```
node scripts/phoenix/alphaaIO.js ${your_account} ${uri}
```
example
```
node scripts/phoenix/alphaaIO.js 0x640a6D5A3f155A8F0636a0396B18Ba5eEdfab440 ipfs://abcxyz
```
